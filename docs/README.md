# Indian Education Loan Calculator

A comprehensive education loan planning tool for Indian students. Calculate EMI, compare scenarios, and make informed financial decisions.

**Live Demo:** https://him001133.github.io/loan/

## Features

### 💰 Main Calculator
- **EMI Calculator** - Instantly calculate monthly education loan repayments in INR
- **Grace Period Analysis** - Understand how moratoriums affect your loan balance
- **Flexible Repayment** - Test interest-only periods and advance payments
- **Visual Charts** - See principal vs interest breakdown
- **Detailed Schedule** - Month-by-month repayment breakdown

### 🔄 Scenario Comparison Tool
- Save up to 3 different loan scenarios
- Compare side-by-side
- Load saved scenarios back into calculator
- Local browser storage (no data transmission)

### 🕐 Timezone Converter
- Convert time between 380+ timezones
- Live reference times for major cities
- Perfect for coordinating across regions

### 📄 Information Pages
- About Us
- Privacy Policy
- Terms of Service
- Contact & Support

## Run Locally

Open [index.html](index.html) in a browser, or serve with a static server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000/

## Deploy on GitHub Pages

1. Push this repository to GitHub
2. Go to Settings → Pages
3. Select "Deploy from a branch"
4. Choose "main" branch and "root" folder
5. Your site will be live at `https://YOUR_USERNAME.github.io/loan/`
