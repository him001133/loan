# Indian Education Loan Calculator

This repository contains a simple, static education loan calculator that can be hosted on GitHub Pages.

## Features
- Calculates monthly EMI in INR
- Estimates total interest and total repayment amount
- Works without any build step or dependencies

## Run locally
Open [index.html](index.html) in a browser, or serve the folder with a simple static server such as:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000/.

## Publish on GitHub Pages
1. Push this repository to GitHub.
2. Open the repository settings.
3. Under Pages, choose the main branch and the root folder.
4. Your site will be available at your GitHub Pages URL.
